لإظهار الفيديوهات في صفحة التمارين، حدث ملف videos.json في هذا المجلد:
- أضف كائنًا لكل فيديو: { "title": "اسم الفيديو", "file": "example.mp4", "thumbnail": "اختياري.png" }
- مثال موجود في videos.json.

ملاحظات:
- يتم تشغيل الفيديو بشكل مكتوم افتراضيًا ويمكنك تشغيل الصوت عبر زر في الصفحة.
- تحت الفيديو يوجد مساحة لكتابة وصف الحركة أو ملاحظاتك.
