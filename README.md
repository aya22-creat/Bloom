# HopeBloom

An AI-powered breast health companion built with Vite, React, TypeScript, Tailwind CSS, and shadcn-ui.

## Getting Started

Prerequisites:

- Node.js 18+
- npm (or pnpm/bun)

Install and run:

```sh
npm install
npm run dev
```

To build for production:

```sh
npm run build
npm run preview
```

## Tech Stack

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn-ui

## Project Structure

- `src/` – application code
- `public/` – static assets
- `server/` – optional backend (Express + SQLite)

## Deployment

You can deploy the built `dist/` folder to any static hosting service (e.g., Netlify, Vercel, GitHub Pages). For backend features, deploy the `server/` separately to your preferred Node hosting.
